YSFPeer
====

Overview

## Description
Peer and Bridge to YSFReflectors.

## Requirement
- Golang
- gopkg.in/yaml.v2

## Build/Install
- download
	```
	$ git clone https://github.com/skubota/YSFPeer.git
	```
- build
	```
	$ make
	$ make install
	```
## Usage

- 1.Edit your config
- 2.run
        ```
        $ ./YSFPeer
        ```
- Usage:
        ```
        $ ./YSFPeer -h
Usage of ./YSFPeer:
  -c string
    	Config file (default "YSFPeer.yml")
  -d	Debug flag
        ```
